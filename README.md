# A-Medida Web

Sitio corporativo de A-Medida para reposicionar la marca, explicar servicios, comunicar el método de trabajo y generar oportunidades cualificadas.

## Desarrollo local

```bash
npm install
npm run dev
```

Abre `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run typecheck
npm run lint
npm run build
```
